# Description

A To-Do-List, made with JavaScript and HTML CSS. In this To-Do-List you can **add as many tasks as you want**, 
remove them and **change the page theme to light or dark mode**.

# Main Features
- HTML and CSS
- JavaScript
- Responsive Design
- Light/Dark mode
